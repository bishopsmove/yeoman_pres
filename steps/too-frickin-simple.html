Too Frickin' Simple

Use npm to install yo and its generators.
npm install -g yo
Create your project directory, cwd, then:
npm install generator-[name-here]
Once the install is finished, check the project's README.md to follow any subsequent steps.
  • Yeoman leverages nodejs and npm, extending it to support its own ecosystem
  • Once you have global installed yeoman, you can either locally or globally install a generator.
